## Installation
1st 
```bash
apt update
apt install git
apt install python

```
2nd
```bash
git clone https://github.com/EthicalUniverse/ip-changer.git
```

## Usage

```bash
cd ip-changer
python ip-changer.py
```

## Features
- Redirect all the traffic to TOR network.
- The Ability to change your IP address every [] seconds.
- Show your public ip.
